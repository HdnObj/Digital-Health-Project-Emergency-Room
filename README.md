# ER-Intelligence: Emergency Room Management System

## Project Overview
A web-based system designed to streamline Emergency Room operations. Developed as part of the **Requirements Engineering for Digital Health (SBES240)** course. This project focuses on solving data silos by providing a unified interface for Admins and Doctors.

## Key Features
- **Landing Page:** Professional medical interface with secure access.
- **Role-Based Access Control (RBAC):** Distinct dashboards for Admins and Doctors.
- **Admin Dashboard:** Total hospital oversight (Patients, Staff, Bed availability).
- **Doctor Dashboard:** Individual patient management and clinical data entry.
- **Data Persistence:** Mock data integration using JSON.

## Tech Stack
- HTML5, CSS3, JavaScript (Vanilla)
- JSON for mock database
- FontAwesome for medical iconography
